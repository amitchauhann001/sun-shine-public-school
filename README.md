# Sun Shine Public School Portal

A modern, high-performance web portal for Sun Shine Public School, built with React, Node.js, and MongoDB.

## 🚀 Features
- **Dynamic Hero Carousel**: Professional banner with school highlights.
- **Student Achievement Slider**: Showcasing academic and extracurricular success.
- **Admin Dashboard**: Secure management of announcements, gallery, and site content.
- **Contact System**: Direct email integration for school inquiries.
- **Full Gallery**: Responsive photo gallery with real school imagery.

## 🛠️ Stack
- **Frontend**: React (Vite), Framer Motion, Redux Toolkit
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Email**: Nodemailer (Gmail SMTP)
- **Deployment**: GitHub Pages (Static Frontend Demo)

## 📦 Deployment
The frontend is automatically deployed to GitHub Pages using GitHub Actions.
